Roadmap Tiga Fase

Dari pilot 3 bulan ke kabupaten yang sepenuhnya berbasis data.

01
FASE 01 · BULAN 1-3

Pilot Project

Pasang 3-5 titik sensor di lokasi paling kritis. Bangun dashboard awal. Hasilkan laporan kuartal pertama sebagai bukti.

  • 3-5 AWLR / ARR pilot
  • Dashboard BPBD / PUPR
  • Laporan operasional Q1
  • Onboarding operator dinas
02
FASE 02 · BULAN 4-9

Scale ke DAS / Kawasan

Setelah pilot terbukti, perluas cakupan ke satu DAS lengkap atau satu kawasan industri / kecamatan rawan.

  • 15-25 titik sensor terintegrasi
  • SCADA kawasan industri
  • OWQM untuk DLH
  • Dashboard pimpinan daerah
03
FASE 03 · BULAN 10-18

Smart Regency Penuh

Integrasi seluruh dinas ke Smart Regency Command Center. Tulang Bawang sebagai kabupaten dengan keputusan berbasis data real-time.

  • Seluruh DAS Way Tulang Bawang
  • Command Center Kabupaten
  • SOP keputusan berbasis data
  • Showcase nasional · Asta Cita
